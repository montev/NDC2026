# NDC Security 2026

Agenda og notater fra [NDC Security 2026](https://ndcsecurity.com), Oslo – 4–5 mars 2026.

Publisert som statisk nettside med MkDocs: **[montev.github.io/NDC2026](https://montev.github.io/NDC2026)**

---

## Innhold

- Full agenda for begge dager (onsdag og torsdag)
- Direktelenker til YouTube Livestream per rom
- Klar for egne notater per foredrag

## Kjør lokalt

```bash
pip install mkdocs mkdocs-material
mkdocs serve
```

Åpne [http://localhost:8000](http://localhost:8000).

## Publiser til GitHub Pages

```bash
mkdocs gh-deploy
```

## Bidra

Feil i agenda, manglende info eller egne notater? Lag en issue eller send en PR.

## Kilde

Agenda er basert på den offisielle konferanseagendaen fra [ndcsecurity.com](https://ndcsecurity.com).
